Publication GitHub Pages :
1. créer un dépôt simple, par exemple ipp-a4p-v6
2. envoyer tout le contenu du dossier à la racine
3. activer Settings > Pages > Deploy from a branch
4. choisir main et /(root)
5. attendre 1 à 3 minutes
