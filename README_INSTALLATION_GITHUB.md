# IPP A4P V4 — Pack complet GitHub

Ce pack contient la version complète du test IPP prête à être publiée sur un dépôt GitHub Pages.

## Contenu
- index.html
- styles.css
- app.js
- questions_ipp_v3.json
- profiles_36_ipp_v3.json
- phase_items_v3.json
- stress_items_v3.json
- 404.html
- .nojekyll

## Mise en ligne
1. Crée un dépôt GitHub vierge.
2. Dépose tous les fichiers du dossier à la racine du dépôt.
3. Active GitHub Pages sur `main` puis `/(root)`.
4. Ouvre l’URL publiée.

## Points clés de cette version
- 60 vignettes
- 360 propositions
- aucune proposition répétée à l’identique
- formulation contextualisée sport / études / vie quotidienne / travail / orientation
